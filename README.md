# Premium Construction SPA

A modern construction company single-page website built with HTML, CSS, and JavaScript, served by an Express.js server.

## Features

- Single Page Application (SPA)
- Sticky navigation with smooth scrolling
- Animated scroll reveal effects
- Responsive mobile hamburger menu
- Counters, FAQ accordion, gallery, testimonials, calculator, and contact form
- Express.js static server

## Folder Structure

```

```

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
npm start
```

3. Open your browser at `http://localhost:3000`

## Development

- `npm start` - launch the production server
- `npm run dev` - launch with `nodemon` if installed globally

## Notes

- This website is fully frontend-based and does not require a database or CMS.
- All sections are delivered on a single page using smooth navigation.
